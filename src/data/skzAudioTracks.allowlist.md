## Excluded by policy (FYI only)

- Hangul-titled tracks (e.g. 안 NOT, 행복, 맞짱) — unless they have a JYP-issued English title we use that title instead (Thunderous, God's Menu).
- Member solo / 3RACHA mixtapes / SKZ-PLAYER covers.
- SKZ-RECORD originals not released on a numbered album/single.

## Pipeline (post-approval)

1. `scripts/fetchAudioTracks.mjs` reads this file, queries iTunes Search US, picks the best match per row (artist must contain "Stray Kids", album fuzzy-matched).
2. 30-second preview MP3 + 600×600 cover saved to `public/audio-game/`.
3. Generated metadata written to `src/data/audioGameTracks.generated.json`.
4. Game reads from the JSON; daily mode is per-day deterministic, unlimited mode is random with recent-history exclusion.
5. Reveal cadence: 2s (start) → 8s (after 1 miss) → 15s (after 2 misses) → 30s (full preview on win or 3rd miss).
6. Reveal screen surfaces a "Listen on Spotify" search link for the full song — STAYs can finish listening on their own account.
